Advertisement Management lets you administrate all the blog advertisements diretctly from the blog backend.

== Description ==

With the Advertisement Management plugin you can add advertisements to your blog directly from the backend. Information about where you can add advertisements below:

Adspot 1: Below posts title on the front page.

Adspot 2: Below the title when post is open.

Adspot 3: Below post content when post is open.

Adspot 4: Below title when a page is open.

Adspot 5: Below page content when a page is open.

Adspot 6: Below the commentboxes.

Adspot 7: Above the header.

Adspot 8: Below the footer.

You can check out the screenshot if you want to see how Advertisement Management looks like in the backend.

== Installation ==

Installtion is simple and should be done within a minute or so.

1. Upload `advertisementmanagement.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. A new menu named Advertisement Management should now appear in your settings-menu.
4. Put in your advertisementcodes and you are good to go.

== Frequently Asked Questions ==

= Can I really put in any type of adcodes, including Google Adsense? =

Yes, you can. You can type in any kind of CSS / HTML - including adcodes from Google Adsense.

= Who can edit the advertisements in Advertisement Management? =

Only administrators of the blog has acces to the plugin page where you can add / edit adcodes.
